### 혼자 짠 코드 
```
import sys
input = sys.stdin.readline
n = int(input())
l = [list(input.split())for i in range(n)]
l.sort(key = lambda x : (-int(x[1]),int(x[2]),-int(x[3]),x[0]))#여기서 정수형으로 바꿔준다. -> 마지막 이름은 정수처리 해주지 않는다. str가 default
for i in l:
    print(l[0],end="\n")
```
